# Hospital Management System – SQL Project

This project demonstrates an end-to-end Hospital Management System database using MySQL.

## Project Overview
The database is designed to manage hospital operations including:
- Patient details
- Doctor information and schedules
- Appointments
- Treatments
- Billing and payments
- Revenue analysis

## Tables Used
- patients
- doctors
- doctor_schedules
- appointments
- treatments
- bills
- payments

## SQL Concepts Covered
- Database & Table creation (DDL)
- Data insertion (DML)
- Joins (INNER, LEFT)
- Aggregate functions
- GROUP BY & HAVING
- Subqueries
- CASE WHEN
- Window Functions
- CTE (WITH clause)
- Views
- Revenue and business analysis queries

## Tools Used
- MySQL
- MySQL Workbench
